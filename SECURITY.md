# Security Policy

## ⚠️ Important Notice

NETREAPER is a penetration testing tool. It is designed to find and exploit 
vulnerabilities. Please use responsibly and only on systems you have 
authorization to test.

## Reporting a Vulnerability

If you discover a security vulnerability in NETREAPER itself (not in the 
third-party tools it wraps), please report it responsibly.

**Email:** security@offtrackmediaofficial.com

### Please Include:
- Affected files/versions
- Reproduction steps / PoC
- Impact assessment
- Suggested fix (if any)

### Response Timeline
- **Acknowledgment:** Within 2 business days (AWST)
- **Initial Assessment:** Within 5 business days
- **Resolution:** Dependent on severity

## Supported Versions

| Version | Supported |
|---------|-----------|
| 4.x     | ✅ Yes    |
| 3.x     | ⚠️ Critical fixes only |
| < 3.0   | ❌ No     |

## Security Best Practices

When using NETREAPER:
1. Only test systems you have written authorization for
2. Keep the tool updated to the latest version
3. Review logs and output for sensitive data before sharing
4. Use in isolated/controlled environments when possible
